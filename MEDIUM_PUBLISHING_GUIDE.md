## 🚀 HOW TO PUBLISH ON MEDIUM

### **Step 1: Sign Up / Log In**
Go to: https://medium.com/
- Create account or log in
- Click "Write" (top right)

### **Step 2: Create Story**
1. **Title:** Copy from article
   ```
   I Built a Research Scanner That Actually Understands Your Field
   ```

2. **Subtitle:** (optional)
   ```
   Template-driven research discovery tested across 8 domains. 
   From physics to medicine to AI. One system. Any field.
   ```

3. **Copy Content:** 
   - Open `MEDIUM_ARTICLE_v2.md`
   - Copy all text
   - Paste into Medium editor

### **Step 3: Format the Article**

**Headers:**
- Medium auto-formats `##` as H2
- Keep the structure as-is

**Code Blocks:**
- Select code
- Click "`<>`" button (code block)
- Medium will syntax highlight

**Images:**
- Click where you want image
- Click image button
- Upload or paste URL
- Add alt text

**Links:**
- Select text
- Click link button
- Paste: https://github.com/Vincent1949/research-scanner

### **Step 4: Add Graphics**

**Minimum 3 images:**

**Hero Image (top):**
- Create or find a compelling visual
- Research/papers/discovery theme
- 1400 x 700px recommended

**Test Results Chart:**
- Screenshot your results table
- Or create bar chart in Excel/Google Sheets
- Shows the 97.7%, 96%, 86.7% stats

**GitHub Screenshot:**
- Take screenshot of: https://github.com/Vincent1949/research-scanner
- Show README, stars, description
- Paste at end before call-to-action

**How to Add Images:**
1. Click `+` button in left margin
2. Select "Image"
3. Upload your file
4. Add caption/alt text

### **Step 5: Add Tags**

Click "Add tags" at top:
```
Research
Open Source
Machine Learning
Academia
Science
ArXiv
PubMed
Knowledge Management
Python
Semantic Search
```
Maximum 5 tags allowed - choose your best 5!

### **Step 6: Add Story Settings**

Click "..." (three dots) → "Story settings"

**Allow responses:** Yes
**Show in search:** Yes  
**Distribution:** Public
**License:** All rights reserved (or your choice)

### **Step 7: Preview**

Click "Preview" button
- Read through entirely
- Check formatting
- Verify links work
- Test on mobile view

### **Step 8: Publish!**

**Two options:**

**A. Publish Immediately:**
- Click "Publish" button
- Fill in preview text
- Share to Twitter (optional)
- Click "Publish now"

**B. Schedule for Later:**
- Click "..." → Schedule
- Choose date/time
- Save
